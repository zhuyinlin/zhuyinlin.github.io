  • 特殊字符

字符

描述

.

匹配换行符之外的任意单字符

^

匹配输入字符串的开始位置

$

匹配输入字符串的结尾位置

*

前面的子表达式 任意次(包括0)

+

前面的子表达式 至少一次

?

前面的子表达式 0 或 1 次

{n}

n 是一个非负整数。匹配确定的 n 次

{n,}

n 是一个非负整数。至少匹配n 次

{n,m}

匹配 [n, m] 次

两项之间的选择

()

标记字表达式

  • 中括号表达式例子

    [0-9a-zA-Z_] 可以匹配一个数字、字母或者下划线
